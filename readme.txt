------README---------

TangoCMS - Conference Management System (built using MEAN JavaScript stack)

The source contains all the configuration files.

MongoDB and NPM must have been installed, MongoDB must have been configured.
Make sure all the neccessary node modules and bower components are installed before running the application.

MongoDB service should be running before starting the application

In git bash or cmd prompt, navigate to 'Source' folder and execute -
npm cache clean
npm install
bower install
npm start

The application runs on http://localhost:3000
This can be changed in case of conflict by changing the value in app.js

AngularJS v1.2.16
Mongoose 4.4.2
ExpressJS 4.13.4
NodeJS 4.2.4
